## üíª How to View the Project

All final work is consolidated in the `main` branch. To review the exercises, you can follow these steps:

1. Download or clone this repository from the `main` branch.
2. Navigate to the theme folder you want to review (e.g., `array-methods/`, `promises/`, etc.).
3. Open the `index.html` file from that folder in your browser.
4. Open the **Developer Tools** (F12 or Right-click ‚Üí Inspect).
    * To see the **code** for each exercise, go to the **Sources** tab and find the `.js` file (e.g., `Ex1.js`).
    * To see the **result** (the output), go to the **Console** tab.
